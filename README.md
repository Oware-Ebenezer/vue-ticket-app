# ticketapp-vue

Vue + Vite + Tailwind implementation of the Ticket Web App.

## Requirements
- Node 16+
- npm

## Setup
1. `npm install`
2. `npm run dev`

## Session token
Session token stored at `localStorage.ticketapp_session`
